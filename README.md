# React Native Self Posts

Second project to practise react native. New issues mentioned:

- Redux
- React Navigation
- Nested Navigators
- SQLite
- Permissions API

## Usage

1. Clone repository

```sh
https://github.com/PodYapolskiy/rn-self-posts.git
cd rn-self-posts
```

2. Execute program

```sh
expo start
```

3. Test the app using Expo Go

Scan QR or use emulator
